# Sleepingowl File Manager

## Install

Via Composer

``` bash
$ composer require devixar/sleepingowlfilemanager
```

## config/app.php providers array

``` php
Devixar\SleepingowlFileManager\SleepingowlFileManagerServiceProvider::class,
```
